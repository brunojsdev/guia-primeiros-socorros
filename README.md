# 🩺 Guia Vital - Primeiros Socorros

O **Guia Vital** é uma aplicação web responsiva projetada para fornecer orientações rápidas e claras sobre protocolos de primeiros socorros. Este projeto foi desenvolvido como uma demonstração técnica para compor um portfólio profissional, focando em usabilidade, interface moderna (UI) e performance.

## 🚀 Tecnologias Utilizadas

* **HTML5 Semântico:** Estrutura focada em acessibilidade e SEO.
* **Tailwind CSS:** Framework utilitário para estilização rápida, responsiva e personalizada.
* **JavaScript (ES6+):** Lógica de filtragem, renderização dinâmica de componentes e manipulação de DOM.
* **Lucide Icons:** Biblioteca de ícones vetoriais para uma interface intuitiva.
* **Inter Font:** Tipografia moderna focada em legibilidade.

## 🛠️ Funcionalidades

* **Busca Dinâmica:** Localização rápida de protocolos de emergência.
* **Design Responsivo:** Interface totalmente adaptável para dispositivos móveis, tablets e desktops.
* **Avisos de Segurança:** Destaque visual para números de emergência (SAMU 192 e Bombeiros 193).
* **Animações Suaves:** Feedback visual através de animações de 'heartbeat' e transições de hover.

## 🎨 Identidade Visual

O projeto utiliza uma paleta de cores moderna e de alto contraste para garantir a leitura em situações críticas:
* **Fundo:** Roxo Profundo (#4A2B99) e Escuro (#150136).
* **Destaques:** Amarelo Vibrante (#ffdd00) para chamadas de ação e avisos.
* **Sinalização:** Vermelho Coração (#ff3366) para elementos vitais.

## ⚠️ Isenção de Responsabilidade

Este é um **projeto acadêmico**. As informações contidas aqui servem apenas para demonstração de habilidades de desenvolvimento front-end. Em emergências reais, as autoridades competentes (SAMU 192 ou Bombeiros 193) devem ser acionadas imediatamente.

## 🔧 Como rodar o projeto

1. Clone o repositório:
'git clone https://github.com/brunojsdev/guia-vital.git'

2. Abra o arquivo 'index.html' em seu navegador ou utilize a extensão Live Server no VS Code.
